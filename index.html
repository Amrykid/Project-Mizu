Project-Mizu by Amrykid

An attempt at creating a compiler for a simple esoteric programming language I created.

Mizu is my attempt at writing a compiler that targets the .NET framework. In order to do that, I created a simple esoteric syntax for executing mathematical expressions. Mizu supports debugging via Visual Studio and the /debug switch.

Dependencies

.NET Framework 4.0 (Client or Full)
